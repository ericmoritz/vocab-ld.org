Welcome to vocab-ld.org, your free Linked Data vocabulary hosting service.

vocab-ld.org serves to allow you to host your Linked Data vocabularies by simply submitting a pull request to ericmoritz/vocab-ld.org

vocab-ld.org is hosted on S3 and is continously deployed using Travis CI.

We currently only support static file serving but there are plans to support the best practices as defined by W3C
